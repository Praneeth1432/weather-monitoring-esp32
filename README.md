# Weather Monitoring System using ESP32

## 📌 Description
This project monitors weather conditions using ESP32 and sensors.

## 🧰 Components Used
- ESP32
- DHT11 (Temperature & Humidity)
- BMP180 (Pressure)
- Rain Sensor
- LDR (Light sensor)
- LCD (I2C)

## ⚙️ Features
- Real-time weather monitoring
- IoT integration using Blynk
- Displays data on LCD

## 📊 Output
- Temperature
- Humidity
- Pressure
- Rain detection
- Light intensity

## 🛠️ How to Use
1. Connect sensors to ESP32
2. Upload code using Arduino IDE
3. Monitor data on LCD and mobile app
